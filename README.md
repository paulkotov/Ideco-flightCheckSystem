# Ideco-flightCheckSystem
Task for Ideco
Welcome to Flight Check System made specially for Ideco.

System allows you to create, edit and delete flight information. This is single page application, witch helps you to do all in one page.

To start this SPA type: 'npm start'

Application consists of 3 parts: header to add flight info; main, that shows info; and footer for filtering;
